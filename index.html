Podstawy tworzenia stron - krótko i zwięźle

Czym jest HTML?

HTML - to z ang. Hypertext Markup Language. Jest to język tagów (znaczników)

Czym są tagi?

Tag to znacznik, służy do zaznaczania tekstu. Nadaje tekstowi określone z góry znaczenie

Czym są atrybuty?

Atrybuty to dodatkowe informacje, które można przypisać do znacznika. Służą do określenia dodatkowych właściwości znacznika. Niektóre tagi mogą mieć potrzebę umiejscowienia dodatkowej cechy tego tagu. Przykładem może być tag <img>, który ma atrybut src, który określa źródło obrazka, który ma być wyświetlony na stronie.

Czym jest CSS?

CSS to język umożliwiający zmiane z jednego miejsca wyglądu wszystkiego na stronie.CSS to kaskadowe arkusze stylów. CSS pozwala na zmianę kolorów, czcionek, marginesów, paddingów, itp.

Czym jest selektor?

Selektor to element, który pozwala na wybranie elementu na stronie, który ma zostać zmieniony. Selektor może być klasą, id, tagiem, atrybutem, itp.

Jak dołączyc plik CSS na stronie?

Aby dołączyć plik CSS na stronie, należy użyć znacznika <link>, który pozwala na podpięcie zewnętrznego pliku CSS do strony. Wystarczy podać ścieżkę do pliku CSS w atrybucie href.
